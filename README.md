# **Título e Sobre**: 

Gerenciador Acadêmico - Sistema para gestão de alunos e turmas da UFF.

## Como rodar o Backend
1. Clone o repositório.
2. Configure o banco de dados MySQL no 'application.properties'.
3. Execute 'mvn spring-boot:run'.
